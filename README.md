NetSpeedTray zeigt deine aktuelle Download- und Upload-Geschwindigkeit direkt im Infobereich der Windows-Taskleiste an. Das Programm läuft im Hintergrund, misst alle 1,5 Sekunden die Netzwerkauslastung aller aktiven Adapter und zeigt sie im Tooltip des Tray‑Symbols als kbit/s oder Mbit/s an, je nach Auswahl im Rechtsklick-Menü.

​
Funktionen

Hintergrund‑Tray‑App ohne sichtbares Hauptfenster.

​
Live‑Anzeige von Download- und Upload-Speed (kbit/s oder Mbit/s umschaltbar).

Rechtsklick‑Menü mit Einheitenwahl und „Beenden“.

  
Symbol dauerhaft in der Taskleiste anzeigen

Damit das Symbol immer sichtbar ist und nicht hinter dem Pfeil verschwindet:

Rechtsklick auf die Taskleiste → Taskleisteneinstellungen öffnen.
Zum Bereich Infobereich bzw. Taskleistenecken-Symbole / Taskleisteneckenüberlauf scrollen.

In der Liste NetSpeedTray suchen und den Schalter auf Ein bzw. „Immer auf der Taskleiste anzeigen“ stellen. 
Alternativ:
Auf den kleinen Pfeil im Infobereich klicken.
Das NetSpeed‑Symbol mit der Maus aus dem Popup auf die Taskleiste ziehen und dort ablegen; Windows merkt sich diese Position dauerhaft.
